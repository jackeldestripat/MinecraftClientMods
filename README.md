# MinecraftClientMods
## instal·lació
1. Copiar la carpeta 1.12.2 a la carpeta del minecraft (a linux /home/user/.minecraft/mods)
2. Executar el launcher TLauncher-2.75.jar
3. Seleccionar el nom d'usuari. 
4. Versió del mine "Forge 1.12.2"
5. Enter the game
